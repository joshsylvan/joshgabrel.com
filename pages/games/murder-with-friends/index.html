Murder With Friends

In 2018 I took part in the Global Game Jam in Brighton. In a group of five people, we created a game called 'Murder With Friends' in 48 hours.

A game about pitching friends against each other, having one player stuck in a creepy dungeon and the other controlling the environment. The goal of the first player is to survive waves of monsters using a pistol or shotgun while also avoiding traps. The other player can unleash monsters in different rooms and activate the traps that will hurt the player.

The game had lots of moving parts that we had to make functional for the demo.

  • The core game was made in Unity.
  • The Web App for activating traps and monsters was built with React and Redux.
  • A small heroku server to send requests from the web app to the game using web sockets.
  • Art. As none of us were artists we all contributed to some pixel art assets.

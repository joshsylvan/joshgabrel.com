This year (2025) I finally finished a small 2D game engine. After a while learning some C++ I finally have a "finished" prototype.

Big Z's Pizza Delivery, is a tiny platformer about delivering Pizza's, paying rent and defeating your langlord.

After a few Handmade Hero episodes and watching some CMake tutorials I made a prototype of my game engine!

Nerd stuff

Built with C++, CMake, OpenGL, OpenAL, ImGui and SDL3.

GitHub Repository

Architecture

The project is split into core (static lib), src (executable), and pizza_delivery (dll / static lib in release).

Core contains shared functions for the "engine" and game. It is statically linked into the executable and pizza_delivery library.

pizza_delivery contains the game update and render functions. This is built as a standalone library.

The src directory contains the main application executable. In debug mode the application will load in the pizza_game library dynamically. If the game library code is updated while the executable is running, the game library is hot reloaded.
